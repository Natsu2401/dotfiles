# Natsu-dotfiles

mở terminal nhập lệnh

sudo pacman -S --needed base-devel git

git clone https://github.com/Natsu2401/dotfiles

tiếp theo

cd dotfiles

chmod +x setup.sh

./setup


đợi chạy xong thì khởi động lại

chọn môi trường bspwm

phím tắt

win + d : menu app

win + enter : terminal

win + shift + i : menu wifi

win + w : tắt app
